# create-earn-app

LI.FI Earn developer tooling starter.

## Goal

Help builders scaffold a working Earn integration quickly:

- discover vaults
- validate config with `earn doctor`
- generate a Next.js starter
- wire Composer deposit execution
- show user positions

## First Build Slice

1. Define the CLI UX.
2. Add Earn API wrappers.
3. Scaffold the generated app structure.
4. Add a local mock mode for testing.
